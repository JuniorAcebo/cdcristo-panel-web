# 🦷 Clínica Dental – Panel Administrativo

Este repositorio contiene el panel administrativo de una clínica dental, desarrollado con el framework Laravel. La aplicación está orientada exclusivamente al personal autorizado de la clínica (dentistas y administradores), permitiendo gestionar de forma eficiente toda la operativa interna del sistema.

---

## 🛠️ Descripción del Proyecto

La plataforma permite al personal de la clínica:

- Gestionar el registro de pacientes y su información personal.
- Administrar las citas odontológicas mediante un calendario interactivo.
- Controlar el catálogo de servicios odontológicos disponibles.
- Gestionar el equipo de odontólogos registrados en el sistema.
- Revisar y responder mensajes enviados desde el sitio web público.
- Acceder a un dashboard con resumen de la actividad del sistema.

Este proyecto corresponde únicamente a la parte administrativa del sistema, independiente del sitio web público para pacientes.

---

## 🚀 Funcionalidades Principales

- **Autenticación segura** — Login con cifrado de contraseñas y control de acceso.
- **Dashboard** — Panel principal con resumen de actividad de la clínica.
- **Gestión de pacientes** — Crear, editar y eliminar registros (CI, nombre, teléfono, género, seguro dental).
- **Gestión de citas** — Calendario interactivo con registro de citas por paciente, odontólogo y servicio.
- **Gestión de odontólogos** — Alta y administración del personal médico.
- **Gestión de servicios** — Administración del catálogo de tratamientos disponibles.
- **Bandeja de mensajes** — Visualización de consultas enviadas desde el sitio público.
- **Diseño responsivo** — Adaptable a dispositivos móviles y de escritorio.
